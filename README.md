#permainan-tembak 

Apa permainan ini?:
Permainan ini adalah permainan shooter(tembak-tembakan), yaitu game dengan sistem menembak musuh. 
Objektif utama dari game ini adalah menembak alien atau UFO yang berterbangan dan tidak boleh melewati player(tidak dihancurkan)
Cara membikin permainan:
1. Carilah image/foto untuk game. Seperti gambar musuh, peluru, background, dan untuk player nya sendiri.
